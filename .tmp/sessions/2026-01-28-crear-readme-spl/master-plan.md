# Plan Maestro - Crear README.md para Guía SPL

Este plan detalla los pasos para crear un archivo `README.md` informativo y estructurado en español para el repositorio de la Guía de Estudio SPL.

## 1. Análisis del Proyecto
- **Nombre**: Guía de Estudio SPL.
- **Origen**: Basado en apuntes de Iñaqui.
- **Tecnología**: AsciiDoc (Asciidoctor PDF).
- **Contenido**: 9 asignaturas del syllabus oficial de piloto de planeador.
- **Herramientas**: Script de auto-generación `watch_book.sh`.

## 2. Estructura del README.md
- Título principal.
- Descripción corta del propósito.
- Características principales (iconos de "trampa", enfoque práctico vs teórico).
- Estructura del libro (Capítulos 1-9).
- Instrucciones de uso y generación del PDF.
- Requisitos del sistema.

1. [ ] Redactar `README.md` en español.
2. [ ] Validar que los enlaces a los archivos y scripts sean correctos.
